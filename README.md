# Chat-ovaci-aplikace

Aplikace určená k plánování chat, rozpočítávání nákladů, jejich odhadu, určování spacího pořádku, vytváření harmonogramu, rozdělování služeb a možnost klást otázky (vytvářet vlákna).
Aplikace rozděluje uživatele na návštěvníky, administrátory (organizátory chaty) a sysadminy (kontrola všeho, ano, je to těžká práce)
